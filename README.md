## 🔍 Troubleshooting common issues

If you encounter problems, check the items below.

**The music does not change in the game.**
Verify that the Spotify desktop app is open. The mod requires the official client to track your current song. Check that you have an internet connection.

**The application fails to connect.**
Close both the app and Spotify. Reopen Spotify first, then reopen the radio mod. Ensure your firewall permissions allow the application to access the internet.

**The game crashes after launch.**
Update your graphics drivers to the latest version. Disable third-party overlays, such as Discord or Steam overlays, if the game remains unstable.
